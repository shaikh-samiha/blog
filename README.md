## 🔥 Features

- ➕ Add new blog posts
- 🖊️ Edit existing posts
- 🗑️ Delete posts with one click
- 💾 MongoDB Atlas integration
- 🌐 Fully responsive (Mobile + Desktop)
- 🔐 Secured with `.env` for database URI
